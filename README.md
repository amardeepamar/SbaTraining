SBA Training Project Program is a real-time,real-world dynemic project in Asp.net MVC 5 with entityframework 6 and SQL Server. SBA Training means Skilled Birth Attendent Training. This project is for State Health Society Bihar,
SBA Training Project consist of Role based Login authentication, CRUD operation with session Authorised User Management,Employee Management,Staff Nurse Management,User Dashboard Report,All Employee Dashboard Report,
Staff-Nurse Dashboard Report all dashboard report can be download in excel and pdf.
